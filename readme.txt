>>>>>>Projects Library in Vanilla Javascript<<<<<< 

This is personal library of custom small projects made in Vanilla Javascript for common and widely used in-demand functionalities for static websites, the library include the efforts made by us Developers for components such as Image Galleries, Custom Cursors, Sliders, Light/Dark Modes, etc. 

You can feel free to use this  library for any purpose. If you have any questions or need help, please don't hesitate to share.

You may also add your code snippets in this repository, in separate folders with the name of your unique project.
